## 🧠 Featured Project

### [AI Memory Reader](https://github.com/nvwalj/ai-memory-reader)

The native macOS & iOS app for browsing AI agent memory files — **Claude Code, OpenClaw, Codex, Cursor, Gemini, Continue, Aider, GitHub Copilot.**

Auto-detects `~/.claude/CLAUDE.md`, project-level memory files, and all major AI tool directories. Beautiful markdown rendering with sidebar tree, full-text search, in-page find, edit mode with auto-save, URL scheme + CLI for agent integration.
